Adapter TV Spielfilm

  • 1 Adapter TV Spielfilm
    • 1.1 Steckbrief
    • 1.2 Konfiguration
    • 1.3 Aktivierung
    • 1.4 Datenpunkte
    • 1.5 Nutzung in VIS
      • 1.5.1 VIS Widget
      • 1.5.2 CSS Konfiguration

Dieser Adapter liefert das TV Programm aus den RSS-Feeds “Tipps” und “jetzt” des Anbieters tvspielfilm.de. Die Daten werden in JSON Objekte gespeichert, um z.B. in ioBroker.vis verarbeitet zu werden.

Steckbrief
aktuelle Version
Voraussetzungen  -/-
Entwickler  Pix
Stichworte TV, Fernsehen, tvspielfilm, Programm
Github Link
Platform  Javascript/Node.js
Lizenz MIT

Konfiguration

Whitelist

Sollen nur Sendungen eines oder mehrerer bestimmter Sender angezeigt werden, sind diese Sendernamen in die Whitelist einzutragen. Enthält die Whitelist mindestens einen Eintrag, wird die Blacklist igoriert.

Blacklist

Sender, die nicht empfangbar sind oder nicht gelistet werden sollen, können in der Blacklist gespeichert und so bei der Ausgabe übersprungen werden. Die Blacklist wird nur berücksichtigt, wenn die Whitelist leer ist.

Aktivierung

Der Adapter startet alle 5min und liest die RSS-Feeds “Tipps”, “Spielfilm-Highlights”, “heute um 20:15 Uhr”, “heute um 22:00 Uhr” und “jetzt” ein.

Datenpunkte

Folgende Datenpunkte werden vom Adapter bereitgestellt:

tvspielfilm.0.json.heute2015 JSON Tabelle * Programm heute 20:15 Uhr
tvspielfilm.0.json.heute2200 JSON Tabelle * Programm heute 22:00 Uhr
tvspielfilm.0.json.jetzt JSON Tabelle * Programm jetzt
tvspielfilm.0.json.tipps JSON Tabelle * Programm Tipps für den Tag
