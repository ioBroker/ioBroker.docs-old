Homematic All in One Image with ioBroker and piVCCU for TINKERBOARD available

Posted on 25. November 2017 by Homoran — No Comments ↓

There is a brand-new image with ioBroker and a virtualized CCU “piVCCU” available at the download area.

This image requires an elv transceiver module  so the piVCCU with HM-IP support can be used.
